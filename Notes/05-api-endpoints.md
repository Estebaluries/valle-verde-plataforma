# API Endpoints

This document provides details about the API endpoints, including their purpose, request/response formats, and examples.
# API Endpoints

## Auth
POST /api/login  
POST /api/register  

## Listings
GET /api/listings  
POST /api/listings  
PUT /api/listings/:id  
DELETE /api/listings/:id  

## Agents
GET /api/agents  
POST /api/agents  

## Clients
GET /api/clients  
POST /api/clients  

## CRM
GET /api/leads  
POST /api/leads  
PUT /api/leads/:id
